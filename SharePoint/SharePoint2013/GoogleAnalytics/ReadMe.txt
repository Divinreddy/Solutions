== Google Analytics for SharePoint 2013 / Office 365 ==

Project Description
This solution allows you to install the Google Analytics code on all sites inside a SharePoint 2013 / Office 365 site collection. 

By installing this sandbox solution, you can paste and add automatically the Google Analytics code to all your Office 365 sites. It works together with Office 365 publishing sites and also with Office 365 collaboration sites (team sites, blogs, etc.)
The latest release supports SharePoint 2013 or SharePoint 2010 on Office 365.
